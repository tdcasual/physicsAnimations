# 国际化 (i18n) 指南

本文档介绍物理动画演示系统的国际化功能。

## 技术栈

- **Vue I18n 9**: Vue 3 官方国际化插件
- **JSON 格式**: 翻译文件使用 JSON 格式
- **组合式 API**: 使用 `useI18n` 组合式函数

---

## 目录结构

```
src/i18n/
├── index.ts           # i18n 配置和实例
└── locales/
    ├── zh-CN.json     # 简体中文
    └── en.json        # 英文
```

## 使用方法

### 在模板中使用

```vue
<template>
  <!-- 基础使用 -->
  <h1>{{ $t('app.name') }}</h1>

  <!-- 带参数 -->
  <p>{{ $t('catalog.itemsCount', { count: 42 }) }}</p>

  <!-- 嵌套键 -->
  <span>{{ $t('login.error.invalid') }}</span>
</template>
```

### 在脚本中使用

```vue
<script setup lang="ts">
import { useI18n } from 'vue-i18n'

const { t, locale } = useI18n()

// 翻译
const message = t('common.save')

// 带参数
const countMessage = t('catalog.itemsCount', { count: 10 })

// 获取/设置当前语言
console.log(locale.value) // 'zh-CN'
</script>
```

### 切换语言

```typescript
import { setLocale, getLocale } from '@/i18n'

// 切换语言
setLocale('en')

// 获取当前语言
const currentLang = getLocale() // 'en'
```

## 添加新语言

1. 创建翻译文件 `src/i18n/locales/[lang].json`
2. 在 `src/i18n/index.ts` 中导入并注册

```typescript
import fr from './locales/fr.json'

const messages = {
  'zh-CN': zhCN,
  en: en,
  fr: fr,  // 添加新语言
}

export const supportedLocales = [
  { value: 'zh-CN', label: '简体中文' },
  { value: 'en', label: 'English' },
  { value: 'fr', label: 'Français' },  // 添加选项
]
```

## 日期和数字格式化

```vue
<script setup lang="ts">
import { useI18n } from 'vue-i18n'

const { d, n } = useI18n()

// 日期
const now = new Date()
const shortDate = d(now, 'short')  // 2024年1月15日
const longDate = d(now, 'long')    // 2024年1月15日星期一

// 数字
const price = n(1234.5, 'currency')  // ¥1,234.50
</script>
```

## 最佳实践

### Do
- ✅ 将所有用户可见文本放入翻译文件
- ✅ 使用语义化的键名
- ✅ 按功能模块组织翻译
- ✅ 复用通用翻译
- ✅ 为翻译提供上下文注释

### Don't
- ❌ 在模板中硬编码文本
- ❌ 使用过于通用的键名
- ❌ 混合不同语言的翻译
- ❌ 在翻译键中包含逻辑

## 提取和同步翻译

### 手动提取

搜索代码中需要翻译的文本：

```bash
# 查找中文字符
grep -r "[\u4e00-\u9fa5]" src --include="*.vue" --include="*.ts"
```
